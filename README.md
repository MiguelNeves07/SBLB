# Saturn Browser: Leak Beta (SBLB)

⚠️ **Importante: Dependência Obrigatória**

Se ao abrir o navegador você receber um erro de "arquivo não encontrado" ou falha de inicialização (como o erro `0x80070002`), significa que o seu Windows não possui o motor de execução do Chromium instalado.

Para resolver, basta baixar e instalar a dependência oficial da Microsoft pelo link abaixo:

👉 [Baixar o WebView2 Runtime (Microsoft)](https://developer.microsoft.com/pt-br/microsoft-edge/webview2#download)

*Recomenda-se baixar a versão **Evergreen Standalone Installer** ou **Evergreen Bootstrapper**.*
